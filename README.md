# Motor Test Stand PCB

This repository contains the PCB design for a DC motor test stand.

## Features
- Current measurement (INA219 / ACS758)
- Temperature sensing (TMP117)
- Microcontroller interface (Arduino)

## Project Goal
Measure torque, efficiency, and electrical parameters of a DC motor using a generator-based load system.

## Files
- /pcb → PCB layout
- /schematic → circuit design
- /gerbers → manufacturing files
